Every now and then I want to know if I own a game already or if I need to buy it to play. Owning games in different stores/platforms doesn't make this as quick as I would like.

Finally this morning I made a thing for me, so I can do exactly that!

[https://observablehq.com/@pvinis/my-games](https://observablehq.com/@pvinis/my-games)

I can type something in the searchbox and I'll get a result, with the store I own it on.

Things I'd like to have:
- A way to auto-update, or even manual but still mass-update (pull new games from all the stores).

Actually that's it. That would be cool. But for now, I'm very happy with it.
